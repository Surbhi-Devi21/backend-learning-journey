# 🚀  Day 4 - Simple Notes REST API

## 📚 Project Overview

Built a simple Notes REST API using Express.js.

The API allows users to perform CRUD operations on notes.

## Features

- Create a Note
- Get All Notes
- Update a Note
- Delete a Note

## Concepts Covered

- REST API
- CRUD Operations
- Express Routing
- Request Body
- Express Middleware
- JSON Handling
- Postman Testing
- HTTP Methods
- API Responses

## Endpoints

| Method | Endpoint | Description |
|---------|----------|-------------|
| POST | /notes | Create a Note |
| GET | /notes | Get All Notes |
| PUT | /notes/:id | Update a Note |
| DELETE | /notes/:id | Delete a Note |

## Tech Stack

- Node.js
- Express.js
- Postman

## Future Improvements

- MongoDB Integration
- Input Validation
- Error Handling
- Authentication

## Outcome

Successfully built a simple CRUD API and tested all endpoints using Postman.
