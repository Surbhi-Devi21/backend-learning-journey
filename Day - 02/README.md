# 🚀Day 02 - Express Routing and Response

## 📌 Overview

In this lesson, I learned how Express handles routes and how a server responds to client requests. I created multiple routes using the GET method and understood the basics of request-response handling.

---

## 📚 Topics Covered

- Express Routing
- GET Requests
- Creating Home Route (`/`)
- Creating Custom Routes
- Sending responses using `res.send()`
- Understanding the Request-Response Cycle
- Basic Deployment

---

## 🛠️ Tech Stack

- Node.js
- Express.js

---

## ▶️ How to Run

```bash
npm install
node index.js
```

Or

```bash
npx nodemon index.js
```

## 📖 What I Learned

- How routing works in Express.
- Difference between URL routes.
- How clients send requests to a server.
- How servers send responses back.
- How to create simple API endpoints.
